循环神经网络

nlp领域并不可少的一个网络

Embedding是神经网络应用到nlp领域一个关键的步骤，他可以把词语转换为数字，从而让神经网络可以读取。

embedding与变长输入处理


使用one-hot编码的embedding。在nlp领域embedding使用很广泛的是dense embedding，词语对应的索引不再是稀疏向量，而是密集向量。

处理变长输入的方法


长度不够填充0。长度过长截断。

导入数据 imdb电影评价数据

imdb = keras.datasets.imdb
vocab_size = 10000
index_from = 3
(train_data, train_labels), (test_data, test_labels) = imdb.load_data(
    num_words = vocab_size, index_from = index_from)

查看数据

# 查看第一个样本
print(train_data[0], train_labels[0])
print(train_data.shape, train_labels.shape)
print(len(train_data[0]), len(train_data[1]))


对于每一个train data，每一个样本都是一个向量，对于train label来说，每一个样本对应一个值，他们的shape是25000，train data，第二维度不定长，所以没有给出，
可以看出第一，第二样本是不定长的，一个为218，一个为189.

print(test_data.shape, test_labels.shape)

查看词表

word_index = imdb.get_word_index()
print(len(word_index))
print(word_index)


词表大小是八万八，key-val的形式，key对应具体词语，val对应词语id，上面设置idex from是3，所以词语偏移需要3。

词语id偏移3，变成对的id。

word_index = {k:(v+3) for k, v in word_index.items()}

特殊搜索索引

word_index['<PAD>'] = 0
word_index['<START>'] = 1
word_index['<UNK>'] = 2
word_index['<END>'] = 3
# 反向word - index
reverse_word_index = dict(
    [(value, key) for key, value in word_index.items()])
# 把train data里面的数字组合成文本
def decode_review(text_ids):
    return ' '.join(
        [reverse_word_index.get(word_id, "<UNK>") for word_id in text_ids])
decode_review(train_data[0])


成功解析为文本！

数据补全

#长度低于500的会被补全，高于500的会被截断
max_length = 500
train_data = keras.preprocessing.sequence.pad_sequences(
    train_data, # list of list
    value = word_index['<PAD>'], #添加上面定义pad 的值
    padding = 'post', # post把padding放在后面, pre
    maxlen = max_length)

test_data = keras.preprocessing.sequence.pad_sequences(
    test_data, # list of list
    value = word_index['<PAD>'],
    padding = 'post', # post, pre
    maxlen = max_length)

print(train_data[0])


第一样本运行结果，添加的是PAD对应的值。

定义模型

embedding_dim = 16
batch_size = 128
model = keras.models.Sequential([
    # 1. define matrix: [vocab_size, embedding_dim]
    # 2. [1,2,3,4..], max_length * embedding_dim
    # 3. batch_size * max_length * embedding_dim
    keras.layers.Embedding(vocab_size, embedding_dim,
                           input_length = max_length),
    #合并                           
    # batch_size * max_length * embedding_dim 
    #   -> batch_size * embedding_dim
    keras.layers.GlobalAveragePooling1D(),
    keras.layers.Dense(64, activation = 'relu'),
    # 二分类使用激活函数sigmoid
    keras.layers.Dense(1, activation = 'sigmoid'),
])
model.summary()
model.compile(optimizer = 'adam', loss = 'binary_crossentropy',
              metrics = ['accuracy'])


160000 = vocab_size * embedding_dim = 100000 * 16
1088 = 64* 16+ 64（w*x + b = 64 * embedding_dim + 64）
65 = 64 * 1 + 1

训练模型

history = model.fit(train_data, train_labels,
                    epochs = 30,
                    batch_size = batch_size,
                    validation_split = 0.2)

学习曲线

def plot_learning_curves(history, label, epochs, min_value, max_value):
    data = {}
    data[label] = history.history[label]
    data['val_'+label] = history.history['val_'+label]
    pd.DataFrame(data).plot(figsize=(8, 5))
    plt.grid(True)
    plt.axis([0, epochs, min_value, max_value])
    plt.show()

plot_learning_curves(history, 'accuracy', 30, 0, 1)
plot_learning_curves(history, 'loss', 30, 0, 1)



训练集的loss一直在下降，而验证集loss下降一定程度后后又上升，说明有过拟合的问题。

循环神经网络




rnn特点：
1.维护一个状态，这个状态可以当前步传给下一步，再传给下一步。
2.对于每一个计算，参数是共享的。
这两个特点，是的rnn可以处理变长数据，不管序列有多长，每个计算都是两个输入。

在做预测的时候，计算的方法跟训练是类似的，做下一步的时候，需要上一步的输出结果进行输入，还有风险就是，如果开头预测错了，后面也会错。

用rnn做文本分类，输入是一个句子，输出是一个二分类的概率，跟之前不一样的是不用每一步都做合并，只需在最后一步进行输出，来作为mlp的输入，因为我们在rnn维护了一个state（状态），这个状态包含了句子的所有信息，在最后一个状态就有整个句子的信息的，这里连接是双向的，意味着这个rnn是双向的神经网络，多维护一个状态，一个从开始到最后，一个从最后到开始，使得每个词的embedding具有上下文信息。
